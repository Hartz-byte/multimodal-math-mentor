# Trigonometric Identities (Algebra)

## Overview
Trigonometric identities are equations involving trigonometric functions that are true for all values of the variables. They're essential for simplifying expressions and solving trigonometric equations.

## KEY FORMULAS

### Fundamental Identities
$$\sin^2(\theta) + \cos^2(\theta) = 1$$
$$1 + \tan^2(\theta) = \sec^2(\theta)$$
$$1 + \cot^2(\theta) = \csc^2(\theta)$$

### Sum and Difference Formulas
$$\sin(A \pm B) = \sin A \cos B \pm \cos A \sin B$$
$$\cos(A \pm B) = \cos A \cos B \mp \sin A \sin B$$
$$\tan(A \pm B) = \frac{\tan A \pm \tan B}{1 \mp \tan A \tan B}$$

### Double Angle Formulas
$$\sin(2\theta) = 2\sin(\theta)\cos(\theta)$$
$$\cos(2\theta) = \cos^2(\theta) - \sin^2(\theta) = 2\cos^2(\theta) - 1 = 1 - 2\sin^2(\theta)$$
$$\tan(2\theta) = \frac{2\tan(\theta)}{1 - \tan^2(\theta)}$$

### Product-to-Sum Formulas
$$\sin A \sin B = \frac{1}{2}[\cos(A-B) - \cos(A+B)]$$
$$\cos A \cos B = \frac{1}{2}[\cos(A-B) + \cos(A+B)]$$
$$\sin A \cos B = \frac{1}{2}[\sin(A+B) + \sin(A-B)]$$

## SOLUTION PATTERN

1. **Identify applicable identity**
2. **Substitute carefully**
3. **Simplify step by step**
4. **Verify result** by checking special values

## EXAMPLES

### Example 1: Simplifying
**Problem**: Simplify $\frac{\sin^2(x)}{1 - \sin^2(x)}$

**Solution**:
- $1 - \sin^2(x) = \cos^2(x)$
- $\frac{\sin^2(x)}{\cos^2(x)} = \tan^2(x)$

### Example 2: Proving Identity
**Problem**: Prove $\sin(A+B) + \sin(A-B) = 2\sin A \cos B$

**Solution**:
- LHS = $(\sin A \cos B + \cos A \sin B) + (\sin A \cos B - \cos A \sin B)$
- $= 2\sin A \cos B$ = RHS ✓

## WATCH OUT

❌ **Wrong**: Using identities without domain restrictions
✅ **Correct**: Remember denominators cannot be zero

❌ **Wrong**: Incorrect sign in formulas
✅ **Correct**: Double-check sum/difference and ± placement

❌ **Wrong**: Forgetting alternate forms
✅ **Correct**: Know all three forms of $\cos(2\theta)$
