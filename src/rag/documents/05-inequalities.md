# Inequalities (Algebra)

## Overview
Inequalities are mathematical statements comparing quantities using <, >, ≤, ≥. They're crucial in optimization problems, domain restrictions, and range analysis.

## KEY FORMULAS

### Linear Inequality
$$ax + b > c \quad \text{or} \quad ax + b < c$$

### Quadratic Inequality
$$ax^2 + bx + c > 0 \quad \text{or} \quad ax^2 + bx + c < 0$$

### Properties
1. Adding/subtracting same number: direction unchanged
2. Multiplying/dividing by positive number: direction unchanged
3. **Multiplying/dividing by negative number: direction reverses** ⚠️

## SOLUTION PATTERN

1. **Rearrange** to standard form
2. **Find critical points** (roots)
3. **Create sign chart**
4. **Test intervals** between critical points
5. **Write solution** in interval notation

## EXAMPLES

### Example 1: Linear Inequality
**Problem**: Solve $3x - 5 > 7$

**Solution**:
- $3x > 12$
- $x > 4$
- **Solution**: $(4, \infty)$

### Example 2: Quadratic Inequality
**Problem**: Solve $x^2 - 5x + 6 < 0$

**Solution**:
- Factor: $(x-2)(x-3) < 0$
- Roots: $x = 2, 3$
- Sign chart: $(x-2)$ and $(x-3)$ both negative when $2 < x < 3$
- **Solution**: $(2, 3)$

## CONSTRAINTS
- Closed intervals for ≤, ≥
- Open intervals for <, >

## WATCH OUT

❌ **Wrong**: Not reversing inequality when multiplying by negative
✅ **Correct**: Always reverse when multiplying/dividing by negative number

❌ **Wrong**: Forgetting to include endpoints for ≤, ≥
✅ **Correct**: Use $[$ and $]$ for closed intervals

❌ **Wrong**: Combining intervals incorrectly
✅ **Correct**: Use ∪ for union of disjoint intervals
